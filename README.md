# data-analyst-nanodegree
Projects completed as part of Udacity's Data Analyst Nanodegree program (2015-2016).

## Projects

### [nyc-subway-ridership-prediction](./nyc-subway-ridership-prediction)
Predicting hourly traffic in New York subways using linear regression.

*Original repository: [github.com/burnssa/nd_project_1](https://github.com/burnssa/nd_project_1)*

### [openstreetmap-mongodb-analysis](./openstreetmap-mongodb-analysis)
Extracting insights from Open Street Map data with MongoDB.

*Original repository: [github.com/burnssa/nd_project_2](https://github.com/burnssa/nd_project_2)*

### [oakland-crime-analysis-r](./oakland-crime-analysis-r)
Analyzing Oakland crime trends with R and the tidyverse.

*Original repository: [github.com/burnssa/nd_project_3](https://github.com/burnssa/nd_project_3)*

### [enron-fraud-detection](./enron-fraud-detection)
Predicting persons of interest from the Enron dataset with ML classifiers.

*Original repository: [github.com/burnssa/nd_project_4](https://github.com/burnssa/nd_project_4)*

### [oakland-crime-visualization-d3](./oakland-crime-visualization-d3)
Visualizing Oakland crime trends with D3.

*Original repository: [github.com/burnssa/nd_project_5](https://github.com/burnssa/nd_project_5)*
